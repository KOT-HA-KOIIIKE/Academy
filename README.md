Этот скрипт не проверяет статьи на соответствие редполитике.
Скрипт главным образом предназначен для автоматизации рутинных действий, которые отнимают слишком много времени (проверяет альты и скругления у картинок, находит открывающиеся в том же окне ссылки, собирает список меток, проверяет наличие мета-описания и т.д.).
Скрипт можно оптимизировать и кастомизировать, дело ваше.
Все, что нужно для начала работы - вставить ссылку на нужную статью в строку 8 (там есть комментарий в коде, куда вставлять) и нажать Run. Дальше скрипт все сделает сам и выведет результат в review.txt.
